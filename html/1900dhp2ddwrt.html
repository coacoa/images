WXR-1900DHP2 / DD-WRT NAS 復活手順書

このメモは、WXR-1900DHP2にDD-WRTを導入し、USBストレージをNAS化するための最短ルートを記録したものです。

1. 物理接続の鉄則

  • 電力問題: バスパワーHDDはスピンアップ時に電力を食うため、認識しない場合はセルフパワーUSBハブを介すること。
  • 認識確認: ls -l /dev/sd* で /dev/sda1 が見えれば物理層はクリア。

2. フォーマットとマウント (SSH操作)

DD-WRTのカーネル(4.4系)で最も安定するのは ext3 です。

# 1. パーティションをext3で初期化 (データは消去される)
mkfs.ext3 /dev/sda1

# 2. マウントポイント作成
mkdir -p /tmp/mnt/sda1

# 3. 手動マウントテスト
mount -t ext3 /dev/sda1 /tmp/mnt/sda1

# 4. 書き込み権限の開放 (重要!)
chmod 777 /tmp/mnt/sda1

# 5. 確認
df -h

3. GUI設定 (Webインターフェース)

Services > USB:
- Core USB Support: Enable
- USB Storage Support: Enable
- Automatic Drive Mount: Enable
Services > NAS (File Sharing):
- Samba: Enable
- Server Name: DD-WRT-NAS
- Shares Path: /tmp/mnt/sda1
- Name: Public
- Access: Read/Write

4. 再起動対策 (Startup Script)

再起動しても自動でマウントされるよう、Administration > Commands の Startup に以下を貼り付けて Save Startup します。

sleep 10
mkdir -p /tmp/mnt/sda1
mount -t ext3 /dev/sda1 /tmp/mnt/sda1
chmod 777 /tmp/mnt/sda1

5. Windowsからのアクセス

エクスプローラーのアドレスバーに \\192.168.1.235 を入力。

Done. 渋いNASライフを！
