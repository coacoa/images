AP 管理 IP 探索メモ（nmap で発見する方法）

AP（例：Buffalo WHX-1900DHP2）を AP モードで使うと、管理画面の IP が DHCP に出てこないことがある。 その場合、nmap のポートスキャンで 80 番が開いている IP を探すのが最も確実。

1. スキャン対象のネットワークを確認

AP が接続されている LAN のセグメントを確認する。 例：OpenWrt の LAN が 192.168.252.1/24 の場合、スキャン範囲は 192.168.252.0/24。

2. nmap で 80 番ポートをスキャン

nmap -p 80 192.168.252.0/24

結果の中で、80/tcp open と表示された IP が AP 本体。

3. ブラウザでアクセス

見つかった IP にブラウザでアクセスする： 

http://192.168.252.xxx/

4. 補足

  • AP モードの Buffalo 機は DHCP リースに名前を残さないことがある
  • ARP にも出ないことがあるため、nmap が最も確実
  • 無線クライアントも DHCP に出るので、AP 本体と混ざる
  • 80 番が開いている IP が管理画面の IP
