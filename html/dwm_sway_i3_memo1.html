dwm / sway / i3 キーバインド統合リファレンス

【重要】 このリストは dwm標準の操作感（Mod1 = Alt キー基準） に統一するための設定表です。
※もしdwm側で Mod4 (Super / Windows キー) を使用している場合は、以下の Alt を Super に読み替えてください。

1. 統合ショートカットキー対応表

端末（ターミナル）起動 Alt + Shift + Enter Alt + Shift + Enter
ウィンドウを閉じる Alt + Shift + c Alt + Shift + c
dmenu (アプリランチャー) Alt + p Alt + p

メインエリアの横幅調整 (狭 / 広) Alt + h / l Alt + h / l

2. sway / i3 用 設定記述例 (config)

~/.config/sway/config または ~/.config/i3/config に以下を貼り付けるか、既存の部分を書き換えてください。

# --------------------------------------------------
#dwmスタイルへの寄せ設定
# --------------------------------------------------
set $mod Mod1

# 基本操作
bindsym $mod+Shift+Return exec alacritty  # 使用端末に書き換えてください
bindsym $mod+Shift+c kill
bindsym $mod+p exec dmenu_run

# フォーカス移動・ウィンドウ移動 (j, k)
bindsym $mod+j focus next
bindsym $mod+k focus prev
bindsym $mod+Shift+j move down
bindsym $mod+Shift+k move up

# 横幅の調整 (dwmの Alt + h / l を再現)
bindsym $mod+h resize shrink width 5 px or 5 ppt
bindsym $mod+l resize grow width 5 px or 5 ppt

# レイアウト切り替え
bindsym $mod+t layout toggle splitv splith
bindsym $mod+f floating toggle
bindsym $mod+m layout stacking

# マルチモニター（画面間移動：カンマとピリオド）
bindsym $mod+comma focus output left
bindsym $mod+period focus output right
bindsym $mod+Shift+comma move workspace to output left
bindsym $mod+Shift+period move workspace to output right

# 終了
bindsym $mod+Shift+q exit
