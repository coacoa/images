📹 WN-AC1600DGR3 + C615 監視システム

1. 【映像】ブラウザで確認 (MJPEG)

ルーター側で mjpg_streamer が動いている前提：

ブラウザで映像を開く (http://192.168.1.2:8080)

2. 【音声】リアルタイム試聴 (NixOS側で実行)

FFmpeg 7.1.1 準拠の最終成功コマンド：

ssh root@192.168.1.2 "arecord -D hw:0,0 -f S16_LE -r 16000 -t raw" | ffplay -f s16le -ar 16000 -ac 1 -i - ※パスワード入力後、Recording... と出れば音が鳴ります。

3. 【録音】5秒間のテスト録音 (ルーター側)

マイクの調子が悪い時の切り分け用：

arecord -D hw:0,0 -f S16_LE -r 16000 -d 5 /tmp/test.wav && ls -lh /tmp/test.wav

4. 【保守】LuCI(管理画面)が壊れた時

opkg update && opkg install luci-base ucode luci-lua-runtime
