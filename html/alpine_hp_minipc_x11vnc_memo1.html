■ 1. sudo を使えるようにする（必要な場合）

su -
apk add sudo
adduser hoge wheel
visudo   # %wheel ALL=(ALL) ALL を有効化

■ 2. x11vnc をインストール

sudo apk add x11vnc

■ 3. VNC パスワードを作成

su -
x11vnc -storepasswd /etc/x11vnc.pass

■ 4. LightDM の Xauthority を確認

ps aux | grep X
# → /var/run/lightdm/root/:0 が auth ファイル

■ 5. x11vnc の OpenRC サービスファイルを作成

/etc/init.d/x11vnc を作成：

#!/sbin/openrc-run

command="/usr/bin/x11vnc"
command_args="-display :0 -auth /var/run/lightdm/root/:0 -rfbauth /etc/x11vnc.pass -forever -shared -noxdamage -repeat -rfbport 5900"
pidfile="/run/x11vnc.pid"
name="x11vnc"
description="x11vnc server"
権限付与： 
chmod +x /etc/init.d/x11vnc

■ 6. サービス起動 & 自動起動登録

rc-service x11vnc start
rc-update add x11vnc

■ 7. Remmina から接続

  • Protocol: VNC
  • Host: 192.168.1.6:5900
  • Password: /etc/x11vnc.pass で設定したもの

■ 補足

LightDM 環境では -auth guess は失敗するため、
/var/run/lightdm/root/:0 を明示指定する必要がある。
