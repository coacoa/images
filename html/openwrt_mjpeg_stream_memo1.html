OpenWrt カメラ配信 復旧メモ

停電後のトラブル解決策：
1. パスワードを求められる → /etc/config/mjpg-streamer の username/password を消す。
2. カクカクする（重い） → option yuv '0' にしてCPU負荷を下げる。

1. 現在の状態を確認するコマンド

CPU負荷を確認（%CPUが90%超えていたらNG） 
top
実行中のフルコマンドを確認 
cat /proc/$(pgrep mjpg_streamer)/cmdline | tr '\0' ' '

2. 修正すべき設定ファイル

vi /etc/config/mjpg-streamer を編集し、以下の「理想的な設定」に合わせる。

config input 'uvc'
    option device '/dev/video0'
    option fps '15'
    option resolution '640x480'
    option yuv '0'             # 0にすることでCPU負荷激減
    option quality '80'

config output 'http'
    option port '8080'
    option www '/www/webcam'
    # option username 'root'   # 認証不要ならコメントアウト
    # option password 'pass'   # 認証不要ならコメントアウト

3. 反映・自動起動コマンド

# 設定反映
/etc/init.d/mjpg-streamer restart

# 自動起動を有効化
/etc/init.d/mjpg-streamer enable

4. 視聴用URL

VLCやブラウザで以下を入力：

http://[OpenWrtのIPアドレス]:8080/?action=stream
