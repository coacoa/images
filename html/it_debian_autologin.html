Debian 13 (Cinnamon) セットアップメモ

1. ifconfig を使えるようにする

Debian 13ではデフォルトで入っていないため、手動でインストールします。

sudo apt update
sudo apt install net-tools

※パスが通っていない場合は /usr/sbin/ifconfig で実行。

2. SSH サーバーの有効化

sudo apt install openssh-server
sudo systemctl enable --now ssh

3. 自動ログインの設定 (LightDM)

設定ファイル /etc/lightdm/lightdm.conf を直接いじるより、専用ディレクトリにファイルを作るのが確実です。

① 設定ファイルの作成

sudo mkdir -p /etc/lightdm/lightdm.conf.d
sudo nano /etc/lightdm/lightdm.conf.d/autologin.conf

以下の内容を記述（hoge は実際のユーザー名に変更）：

[Seat:*]
autologin-user=hoge
autologin-user-timeout=0
autologin-session=cinnamon

② 既存ファイルの掃除（重要）

/etc/lightdm/lightdm.conf の中に [LightDM] セクションで自動ログインが書いてあるとエラーになるため、 # でコメントアウトするか削除します。

4. ユーザー権限の付与

システムが自動でログインさせるための権限を追加します。

sudo groupadd -r autologin
sudo gpasswd -a hoge autologin

5. 設定の確認と再起動

設定が正しく読み込まれているか確認：

sudo /usr/sbin/lightdm --show-config

問題なければ再起動：

sudo reboot
