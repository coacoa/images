OpenWrt ネットワーク設定メモ

1. LAN / WAN インターフェース

項目 例 説明
LAN IP 192.168.251.1 / Static ルーター自身の固定IP。LAN 内クライアントのゲートウェイになる。
LAN DHCP Server 有効（Ignore = OFF） LAN 内のクライアントに IP を自動配布する。
WAN IP 192.168.1.200 / DHCP 上位ルーター（F660Tなど）から自動取得。インターネット側への接続用。

2. クライアント IP

項目 例 説明
ノートPC / LAN Wi-Fi 192.168.251.172 / DHCP DHCP による自動取得。MAC 固定で Static Lease 可能。

3. SSH アクセス

場所 接続先 IP 説明
LAN 内 192.168.251.1（OpenWrt） LAN 内なら NAT を気にせずアクセス可能。
WAN 側 192.168.1.x + ポートフォワード 上位ネットワークからアクセスする場合はポートフォワードが必要。

4. ポイントまとめ

  • LAN インターフェースは Static IP にするのが普通。
  • DHCP Server は LAN 上のクライアントに IP 配布する。Ignore = ON なら停止。
  • クライアント IP は DHCP で自動取得。必要なら Static Lease で固定。
  • SSH は LAN 内なら LAN IP で直接接続可能、WAN 側ならポートフォワードが必須。
  • LAN 側ポート（例: 8080, 5900）は firewall や port forward 設定でアクセス制御可能。
