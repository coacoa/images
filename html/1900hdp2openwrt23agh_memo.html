WXR-1900DHP2 構築メモ (OpenWrt 23.05)

基本ネットワーク情報:
ルーターIP: 192.168.235.1 / 24
AGH管理画面: http://192.168.235.1:8080
DNS構成: PC(53) → dnsmasq(53) → AdGuard Home(5353)

1. AdGuard Home 基本設定

項目 設定値
管理画面ポート 8080 (WAN開放済み: 3000/8080)
DNSサーバーポート 5353
クエリログ保持期間 24時間〜3日間 (本体Flash保護のため短め推奨)

2. ネットワーク・DNS設定 (SSH)

dnsmasqの問い合わせをAGHにリレーし、クライアントに自身のIPを通知する設定：

# DNSリレー設定
uci set dhcp.@dnsmasq[0].port='53'
uci set dhcp.@dnsmasq[0].noresolv='1'
uci del_list dhcp.@dnsmasq[0].server
uci add_list dhcp.@dnsmasq[0].server='127.0.0.1#5353'

# LAN内クライアントへのDNS通知
uci del_list dhcp.lan.dns
uci add_list dhcp.lan.dns='192.168.235.1'

# IPv6 DNSバイパス防止
uci set dhcp.lan.ra_dns='0'
uci set dhcp.lan.dns_service='0'

uci commit dhcp
/etc/init.d/dnsmasq reload
/etc/init.d/odhcpd reload

3. ファイアウォール設定 (WANアクセス許可)

# ポート3000(Setup) / 8080(Admin) の開放
uci add firewall rule
uci set firewall.@rule[-1].name='Allow-AGH-WAN'
uci set firewall.@rule[-1].src='wan'
uci set firewall.@rule[-1].dest_port='3000 8080'
uci set firewall.@rule[-1].proto='tcp'
uci set firewall.@rule[-1].target='ACCEPT'

uci commit firewall
/etc/init.d/firewall reload

4. メンテナンスコマンド

  • 設定の完全保存: uci commit && sync
  • AGH再起動: /etc/init.d/adguardhome restart
  • DHCP配布確認: cat /tmp/dhcp.leases
  • 物理リンク確認: ip addr show br-lan
注意: USBメモリを認識していないため、データは全て内部Flash (/overlay) に保存されています。ログが肥大化して容量を圧迫しないよう、AGH側で定期的なログ削除設定を確認してください。
