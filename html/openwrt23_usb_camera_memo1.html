📷 OpenWrt 23.05 WebCam 起動メモ

Logicool C615 を WZR-1900DHP2 で動かすための最短手順です。

前提: カメラが /dev/video0 または /dev/video1 として認識されていること。

Step 1: 門番（ファイアウォール）を開ける

OpenWrt 23.05は fw4 (nftables) なので、以下のコマンドで8080番ポートを開放します。

nft add rule inet fw4 input tcp dport 8080 accept
/etc/init.d/firewall reload

Step 2: 配信サーバーを起動する

C615に最適な「MJPEGモード」かつ「全開放(-b 0.0.0.0)」設定です。

# 念のため古いプロセスを掃除
killall -9 mjpg_streamer

# 起動コマンド（決定版）
mjpg_streamer -i "input_uvc.so -r 640x480 -f 15" -o "output_http.so -p 8080 -b 0.0.0.0 -n" &

Step 3: 映像を確認する

VLC または ブラウザで以下のURLを開きます。

http://192.168.1.2:8080/?action=stream

🔍 映らない時のトラブルシューティング

  • pingは通るが501エラーが出る: サーバーは生きています！URLに ?action=stream を付け忘れていないか確認。
  • (null) エラーが出る: カメラを抜き差しして、ls -l /dev/video* でデバイス番号を再確認。
  • カクカクする: コマンドの -r 640x480 を -r 320x240 に下げると軽くなります。

✨ これでいつでも C615 の映像を楽しめます！ ✨

OpenWrt MJPG Streamer メモ

📷 MJPG Streamer 起動コマンド

USB カメラを 640x480 / 15fps で配信し、HTTP を 8080 で公開する。

mjpg_streamer -i "input_uvc.so -d /dev/video0 -r 640x480 -f 15" \
              -o "output_http.so -p 8080 -l 0.0.0.0 -n" &

🌐 WAN 側から 8080 をアクセス可能にする Firewall ルール

WAN → 8080 を許可する永続ルール。

uci add firewall rule
uci set firewall.@rule[-1].name='Allow-MJPG-WAN'
uci set firewall.@rule[-1].src='wan'
uci set firewall.@rule[-1].dest_port='8080'
uci set firewall.@rule[-1].proto='tcp'
uci set firewall.@rule[-1].target='ACCEPT'
uci commit firewall
/etc/init.d/firewall restart

🔗 アクセス URL

WAN 側からブラウザでアクセス：

http://:8080/?action=stream

例：

http://192.168.1.215:8080/?action=stream
