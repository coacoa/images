OpenWrt RSSポータル構築備忘録

WN-DX1167RなどのOpenWrt環境で、広告なしの爆速ニュースポータルを構築する手順です。

1. 必須パッケージのインストール

HTTPS通信（SSL）を安定させ、日本語の文字化けを防ぐために必要です。

opkg update
opkg install ca-bundle ca-certificates
# もしSSLエラーが出るなら
opkg install libustream-mbedtls

2. RSS取得スクリプトの作成

/usr/bin/get_news.sh という名前で保存します。4スレッドCPUを活かして高速に処理します。

cat << 'EOF' > /usr/bin/get_news.sh
#!/bin/sh
FILE="/www/news.html"

fetch_rss() {
    TITLE=$1; URL=$2
    echo "
$TITLE
" >> $FILE
    wget --no-check-certificate -qO- "$URL" | tr -d '\n' | sed 's/<item/
<item/g' | \
    sed -n 's/.*<title>\([^<]*\)<\/title>.*<link>\([^<]*\)<\/link>.*/<p><a href="\2" target="_blank" style="font-size:1.1em; color:#0055ff; text-decoration:none;">・\1<\/a><\/p>/p' | \
    head -n 6 >> $FILE
}

echo "<html><head><meta charset='utf-8'><meta name='viewport' content='width=device-width,initial-scale=1'><title>Portal</title><style>body{font-family:sans-serif;padding:15px;background:#f0f2f5;color:#333;} .box{background:#fff;padding:15px;margin-bottom:15px;border-radius:10px;box-shadow:0 2px 5px rgba(0,0,0,0.1);} h1{font-size:1.4em;color:#1a73e8;text-align:center;} h3{border-left:5px solid #1a73e8;padding-left:10px;margin:15px 0 10px 0;font-size:1.1em;}</style></head><body>" > $FILE
echo "<h1>WN-DX1167R Home Station</h1>" >> $FILE

# ☀️ 天気
echo "<div class='box'><h3>☀️ 関東の週間天気</h3>" >> $FILE
wget --no-check-certificate -qO- "https://www.drk7.jp/weather/xml/13.xml" | tr -d '\n' | sed 's/<pref/
<pref/g' | grep "東京都" | sed 's/<day/
<day/g' | sed -n 's/.*date="\([^"]*\)".*<telop>\([^<]*\)<\/telop>.*<low>\([^<]*\)<\/low>.*<high>\([^<]*\)<\/high>.*/<p style="margin:5px 0;"><b>\1<\/b>: \2 (\3℃\/\4℃)<\/p>/p' | head -n 7 >> $FILE
echo "</div>" >> $FILE

# 🏎️ モータースポーツ
echo "<div class='box'>" >> $FILE
fetch_rss "🏎️ autosport web" "https://www.as-web.jp/feed"
fetch_rss "🏁 Motorsport.com" "https://jp.motorsport.com/rss/f1/news/"
echo "</div>" >> $FILE

# 💻 IT・ガジェット
echo "<div class='box'>" >> $FILE
fetch_rss "💻 ITmedia" "https://rss.itmedia.co.jp/rss/2.0/itmedia_all.xml"
fetch_rss "🚀 Gizmodo" "https://www.gizmodo.jp/index.xml"
echo "</div>" >> $FILE

# 🌍 総合
echo "<div class='box'>" >> $FILE
fetch_rss "🌍 Yahoo News" "https://news.yahoo.co.jp/rss/topics/top-picks.xml"
echo "</div>" >> $FILE

echo "<hr><p style='text-align:center; font-size:0.8em; color:#666;'>Update: $(date)</p></body></html>" >> $FILE
EOF
chmod +x /usr/bin/get_news.sh

3. 自動更新の設定（Cron）

30分ごとに最新ニュースに更新するように設定します。

# LuCIの場合: System -> Scheduled Tasks に以下を貼り付け
*/30 * * * * /usr/bin/get_news.sh

4. 使い方

ブラウザから http://192.168.1.1/news.html を開く。
