Flash(AS3) → 現代HTML / JS 対応メモ

1. MovieClip と div の対応

Flash (AS3)

var mc:MyClip = new MyClip();
mc.x = 100;
mc.y = 50;
addChild(mc);

HTML + JavaScript

<div id="mc"></div>

const mc = document.getElementById("mc");
mc.style.left = "100px";
mc.style.top  = "50px";

2. instance name → id / class

Flash

mc.visible = false;

HTML

mc.style.display = "none";

3. 入れ子構造（addChild）

Flash

parent_mc.addChild(child_mc);

HTML

<div id="parent">
  <div id="child"></div>
</div>

4. イベント（クリック）

Flash

mc.addEventListener(MouseEvent.CLICK, onClick);

function onClick(e:MouseEvent):void {
  trace("clicked");
}

HTML

mc.addEventListener("click", () => {
  console.log("clicked");
});

5. enterFrame → requestAnimationFrame

Flash

addEventListener(Event.ENTER_FRAME, loop);

function loop(e:Event):void {
  mc.x += 1;
}

HTML

function loop() {
  x += 1;
  mc.style.left = x + "px";
  requestAnimationFrame(loop);
}
loop();

6. timeline が無い代わりに

  • CSS animation
  • JavaScript 状態管理
  • requestAnimationFrame
@keyframes move {
  from { transform: translateX(0); }
  to   { transform: translateX(200px); }
}

7. TypeScript（AS3 に近い書き味）

class Box {
  x: number = 0;
  el: HTMLElement;

  constructor(el: HTMLElement) {
    this.el = el;
  }

  move() {
    this.x += 1;
    this.el.style.left = this.x + "px";
  }
}

まとめ（Flash脳での理解）

  • div = MovieClip
  • id = instance name
  • class = 見た目や役割
  • timeline は自分で作る
  • JS / TypeScript が AS3 の後継
