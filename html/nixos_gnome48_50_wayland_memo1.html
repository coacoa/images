NixOS 25.05 → 26.05 アップグレード後の GNOME 50 カーソル不具合修復メモ

環境

  • NixOS 25.05 → 26.05 にアップグレード
  • GNOME 50
  • Wayland
  • NVIDIA GTX 1050
  • NVIDIA Driver 580.142

症状

  • GNOMEログイン後、マウスカーソルが真っ白な四角になる
  • 通常の矢印カーソルが表示されない

確認

echo $XDG_SESSION_TYPE

結果:

wayland

修復手順

  1. GNOME Terminal を開く
  2. カーソルテーマを Adwaita に変更する
gsettings set org.gnome.desktop.interface cursor-theme 'Adwaita'

または Bibata を使用する場合:

gsettings set org.gnome.desktop.interface cursor-theme 'Bibata-Modern-Ice'
  1. ログアウト
  2. 再ログイン

確認コマンド

現在のカーソルテーマ:

gsettings get org.gnome.desktop.interface cursor-theme

利用可能なカーソルテーマ:

find /usr/share/icons -name cursor.theme

結果

  • 白い四角カーソルが正常なカーソルへ戻った
  • Waylandセッションのまま利用可能になった

備考

  • GNOME 50ではX11セッションは基本的に利用不可
  • GSK_RENDERER=ngl は今回の症状には不要だった
  • 原因はカーソルテーマ関連の不整合の可能性が高い
