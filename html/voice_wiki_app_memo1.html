NixOS Voice Wiki Project Memo

目的

寝る前にWikipediaを音声で調べたり読み上げたりできる軽量システムを作る。

基本方針

  • NixOS上で動作
  • 普段は何もインストールしない
  • nix develop 実行時だけ利用可能
  • GPU不要
  • 古いPCでも動作可能

最初の構成

Spaceキー押下
↓
録音開始
↓
Spaceキー解放
↓
音声認識(Vosk)
↓
Wikipedia検索
↓
記事取得
↓
Piperで読み上げ

採用候補

  • Vosk（音声認識）
  • Piper TTS（音声合成）
  • Wikipedia API
  • Textual（TUI）

TUIイメージ

┌─────────────────────────────┐
│ WIKI VOICE TERMINAL         │
├─────────────────────────────┤
│ 状態: READY                 │
│                             │
│ Space: 押しながら話す       │
│ Esc: 終了                   │
└─────────────────────────────┘

音声コマンド案

  • 長く
  • 短く
  • 関連
  • 停止
  • 続き

ログ保存構想

logs/
├── 2026-06-01.md
├── 2026-06-02.md
└── ...

記録内容例:

質問: 徳川家康
検索: 徳川家康
要約: 江戸幕府を開いた武将...
