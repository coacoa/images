OpenWrt 23.05 extroot 設定手順

Aterm WG1600DGR3 などの内蔵ストレージが少ないルーターで、USBメモリ (32GB/ext4) を使用してシステム容量を拡張した際の記録です。

1. 必要パッケージの導入

opkg update
opkg install kmod-usb-storage kmod-usb-storage-uas kmod-fs-ext4 block-mount

2. データの同期 (Overlayの移行)

内蔵フラッシュの /overlay データを USBメモリ (/dev/sda1) へコピーします。

mkdir -p /tmp/cproot
mount /dev/sda1 /tmp/cproot
tar -C /overlay -cvf - . | tar -C /tmp/cproot -xf -
umount /tmp/cproot

3. マウント設定 (fstab) の編集

block detect > /etc/config/fstab を実行後、vi /etc/config/fstab で以下のように調整します。

修正後の /etc/config/fstab 例

config 'global'
    option anon_mount '0'
    option anon_swap '0'
    option auto_mount '1'
    option auto_swap '1'
    option check_fs '0'
    option delay_root '5' # USB認識待ち時間を追加

config 'mount'
    option target '/overlay' # ここを /overlay に
    option uuid '1324cad3-14cf-4eca-933c-111111e25b01'
    option enabled '1' # ここを 1 に

4. 確認方法

再起動後、df -h を実行し、/ または /overlay のサイズが拡張されていれば成功です。

# 成功例の表示
Filesystem                Size      Used Available Use% Mounted on
/dev/sda1                27.9G      5.0M     26.4G   0% /overlay
overlayfs:/overlay       27.9G      5.0M     26.4G   0% /
注意: 運用中にUSBメモリを抜かないでください。システムがハングします。
