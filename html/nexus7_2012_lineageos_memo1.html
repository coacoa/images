Nexus 7 (2012, grouper) 復活メモ

このメモは、Nexus 7 を LineageOS 14.1 で復活させるまでの実際の手順と、特に重要だったポイントをまとめたもの。

1. Fastboot に入れない問題

最初にぶつかった最大の問題は、電源 + 音量下 を何度やっても Android が起動してしまい、fastboot に入れなかったこと。

原因は Nexus 7 特有の癖で、以下の方法でしか fastboot に入れなかった：

● 決め手になった方法（成功率100%）

  1. Android が起動している状態で adb reboot を実行
  2. 画面が真っ暗になった「瞬間」に 音量下ボタンを押しっぱなし
  3. Google ロゴが出ても離さない

→ これで fastboot に入れた。

● なぜ電源 + 音量下ではダメだったのか

  • Nexus 7 は電源 + 音量下の長押しが非常にシビア
  • バッテリー劣化個体では成功率が極端に低い
  • adb reboot → 再起動割り込みの方が確実

2. fastboot で TWRP を焼く

fastboot に入れたら、PC から以下を実行：

fastboot flash recovery twrp-3.7.0_9-0-grouper.img

ログが以下なら成功：

Sending 'recovery' (10838 KB) OKAY
Writing 'recovery'          OKAY

3. fastboot で上下ボタンが効かない問題

fastboot 画面で音量上下が効かず、Recovery Mode が選べなかった。

● 原因

USB ケーブルを挿したままだとボタンが効かなくなる個体がある。

● 解決方法

  1. USB ケーブルを抜く
  2. 音量上下が復活する
  3. Recovery Mode を選ぶ
  4. TWRP 起動後に USB を挿し直す

4. LineageOS ZIP を push する

TWRP が起動したら PC で：

adb push lineage-14.1-20211120-UNOFFICIAL-grouper.zip /sdcard/

その後 TWRP で：

  1. Wipe → Advanced Wipe → System / Data / Cache
  2. Install → ZIP を選ぶ
  3. Swipe to flash
  4. Wipe Dalvik / Cache
  5. Reboot → System

5. 復活後の環境

LineageOS 14.1 が正常に起動し、ブラウザで YouTube も視聴可能。

広告ブロックは以下の方法で可能：

  • Firefox + uBlock Origin
  • NewPipe（広告ゼロ）
  • Brave ブラウザ

まとめ

今回の復活で最も重要だったポイント：

  • adb reboot → 再起動の瞬間に音量下（fastboot への唯一の道）
  • USB を抜かないと fastboot のボタンが効かない
  • fastboot reboot recovery は使わない（Nexus 7 は無視する）
  • TWRP は fastboot → Recovery Mode からしか起動しない

この4つを理解すれば、Nexus 7 はいつでも復活できる。

6. wget で TWRP / LineageOS が正しく落ちない問題

Nexus 7 復活中に最も危険だったのが、wget でダウンロードしたファイルが本物ではなく、HTML だったという問題。

● 実際に起きた症状

  • TWRP を wget したら 6.7KB のファイルが落ちてきた（本物は 10.8MB）
  • LineageOS ZIP を wget したら 数MB の偽物が落ちてきた（本物は 299MB）
  • fastboot flash すると当然エラーやフリーズが起きる

● 原因

  • 公式ミラーが HTML を返している（ダウンロードページそのもの）
  • wget がリダイレクト先の HTML を保存してしまう
  • 古い TLS/HTTPS の相性問題
  • User-Agent がブラウザでないため、正しいファイルが返されない

● 偽ファイルの見分け方

file twrp-3.2.3-0-grouper.img
# → HTML document と出たら偽物

またはサイズが明らかに小さい：

  • TWRP → 10MB 以上が正しい
  • LineageOS → 200〜300MB が正しい

● 正しい対処法

  1. ブラウザで直接ダウンロードする（最も確実）
  2. PC に保存してから adb push で端末へ送る
  3. どうしても wget を使う場合は、正しい直リンクを使う

● 実際に成功した wget の例

LineageOS 14.1（grouper）の正しい直リンク：

wget https://mirrorbits.lineageos.org/full/grouper/20211120/lineage-14.1-20211120-UNOFFICIAL-grouper.zip

TWRP（10.8MB）の正しい直リンク：

wget https://dl.twrp.me/grouper/twrp-3.7.0_9-0-grouper.img

※ ただしミラーが落ちていると HTML が返るため、ブラウザで落とすのが最も安全。

● まとめ

  • wget は HTML を誤って保存することがある
  • サイズが小さいファイルは絶対に偽物
  • 本物は TWRP 10MB、LineageOS 299MB
  • ブラウザで落として adb push が最も確実
